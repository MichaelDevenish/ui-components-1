# Icon

## Basic FontAwesome Icon

```jsx
  <FontAwesomeIcon
    type='star'
  />
```

## Basic Intelli Icon

```jsx
  <IntelliIcon
    type='smile'
  />
```

## Large Icon

```jsx
  <FontAwesomeIcon
    type='star'
    isLarge
  />
```

## 4x size Icon

```jsx
  <FontAwesomeIcon
    type='star'
    size={4}
  />
```

## Spinning Icon

```jsx
  <FontAwesomeIcon
    type='star'
    isSpinning
  />
```

## Coloured Icon

```jsx
  <FontAwesomeIcon
    type='star'
    color='rgb(204,204,0)'
  />
```

## Mix and match

Icon props can be used together!

```jsx
  <FontAwesomeIcon
    type='star'
    color='rgb(204,204,0)'
    isSpinning
    size={3}
  />
```
