# Icon

### Basic Icon

```jsx
  <Icon
    type='star'
  />
```
### Large Icon

```jsx
  <Icon
    type='star'
    isLarge
  />
```
  
### 4x size Icon
  
```jsx
  <Icon
    type='star'
    size={4}
  />
```
  
### Spinning Icon
  
```jsx
  <Icon
    type='star'
    isSpinning
  />
```
  
### Coloured Icon
  
```jsx
  <Icon
    type='star'
    color='rgb(204,204,0)'
  />
```

### Mix and match

Icon props can be used together!

```jsx
  <Icon
    type='star'
    color='rgb(204,204,0)'
    isSpinning
    size={3}
  />
```
