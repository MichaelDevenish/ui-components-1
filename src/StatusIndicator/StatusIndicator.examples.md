#### Basic status indicator

```jsx
<StatusIndicator
  text='Approved'
/>
```

#### Status Indicator with subtitle text

```jsx
<StatusIndicator
  text='Approved'
  subtitleText='25 days ago'
/>
```

#### Hollow dot

```jsx
<StatusIndicator
  text='Approved'
  isHollow
/>
```

#### Coloured dot

```jsx
<div>
  <StatusIndicator
    text='Approved'
    subtitleText='25 days ago'
    color='rgb(255, 0, 0)'
  />

  <StatusIndicator
    text='Approved'
    subtitleText='25 days ago'
    color='rgb(255, 0, 0)'
    isHollow
  />
</div>
```

#### Coloured text

```jsx
<StatusIndicator
  text='Approved'
  subtitleText='25 days ago'
  textColor='success'
/>
```
