## Basic label

```jsx
<DotStatusLabel
  text='Hello world!'
/>
```

## Basic label with subtitle text

```jsx
<DotStatusLabel
  text='Hello world!'
  subtitleText='FooBar'
/>
```

## Hollow label

```jsx
<DotStatusLabel
  text='Hello world!'
  isHollow
/>
```

## Coloured text

```jsx
<DotStatusLabel
  text='Hello world!'
  subtitleText='FooBar'
  textColor='success'
/>
```

## Coloured icon

```jsx
<div>
  <DotStatusLabel
    text='Hello world!'
    subtitleText='FooBar'
    color='rgb(128, 0, 96)'
  />

  <DotStatusLabel
    text='Hello world!'
    subtitleText='FooBar'
    color='rgb(0, 96, 128)'
    isHollow
  />
</div>
```
