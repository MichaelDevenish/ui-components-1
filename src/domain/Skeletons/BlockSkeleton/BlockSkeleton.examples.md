#### Default Block Skeleton

```jsx
<BlockSkeleton
  showSkeleton={true}
/>
```

#### Block Skeleton with provided Width and Height
Provide width and height either as a number to be converted to px, or as a string containing the measurement (eg. 50%)
```jsx
<BlockSkeleton
  showSkeleton={true}
  width={'50%'}
  height={200}
/>
```
