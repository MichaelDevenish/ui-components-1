#### Line Skeleton

```jsx
<div>
  <Skeleton
    skeletonOptions={{
      showSkeleton: true,
      shape: 'line'
    }}
  />
  <Skeleton
    skeletonOptions={{
      showSkeleton: true,
      shape: 'line',
      width: 250
    }}
  />
</div>
```
